## About
**SkinCan** is a **libre** skin cancer detection android app, created for Bangkit Product-based Capstone Project.

### MVP (Minimum Viable Product)
- [x] Onboarding
- [x] Auth
- [x] Camera
- [x] Scan Result
- [ ] Result List
- [x] News List (Placeholder)
